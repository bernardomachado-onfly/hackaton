Opção 2: Console do navegador (F12)

Abra app.onfly.com, pressione F12, vá em Console e cole:

Nota: Quando injetado na página da Onfly, o widget captura automaticamente o token de sessão do usuário logado — sem necessidade de autenticação adicional.

Como funciona

  • O script injeta o widget.js na página atual
  • Se estiver no app.onfly.com, captura o token do localStorage
  • O token é enviado ao backend junto com cada mensagem
  • O backend usa o token para buscar voos/hotéis reais na API da Onfly
